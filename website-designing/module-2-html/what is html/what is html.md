# what is html ?

  1. html stands for hyper text markup language
  2. it is used to create web pages for any website
  3. it is a markup language that uses tags to structure and format content on the web
  4. html is the backbone of the web and is essential for building websites and web applications
  5. it allows developers to create and organize content, add images and multimedia, and create links between pages.
  6. html is a standard language that is supported by all web browsers, making it a universal tool for web development.

  7. html is often used in conjunction with other web technologies such as css and javascript to create dynamic and interactive web pages.

  8. html is a constantly evolving language, with new features and updates being added regularly to improve its functionality and capabilities.


  **features of html**

    1. html is a simple and easy-to-learn language that allows developers to create web pages quickly and efficiently.

    2. html is a flexible language that can be used to create a wide variety of web pages, from simple static pages to complex dynamic applications.

    3. html is a widely supported language that is compatible with all web browsers, making it accessible to a large audience.

    4. html allows developers to create structured and organized content using tags, which can be easily styled and formatted using css.


    5. html supports multimedia content such as images, videos, and audio, allowing developers to create rich and engaging web pages.

    6. html allows developers to create links between pages, making it easy for users to navigate and explore a website.



    **html structure**

    1. html documents are structured using a series of tags that define the different elements of the page.

    2. the basic structure of an html document includes a doctype declaration, a head section, and a body section.

    3. the doctype declaration specifies the version of html being used and helps browsers render the page correctly.

    4. the head section contains metadata about the page, such as the title, description, and keywords, as well as links to external resources such as stylesheets and scripts.

    5. the body section contains the main content of the page, including text, images, and other multimedia elements.

    6. html save with its file extension .htm or .html
    
    ```
    <!DOCTYPE html>
    <html>
    <head>
        <title>My First HTML Page</title>
    </head>
    <body>
        <h1>Welcome to My Website</h1>
        <p>This is my first HTML page.</p>
        <img src="image.jpg" alt="My Image">
        <a href="https://www.example.com">Visit Example.com</a>
    </body>
    </html>

    ```
